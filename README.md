## Simple Calculator

### Project Details
- Run a bash application generate a set of mathematical times tables based on input given by the user.

### Built with
 - Vim text editor
 - Powershell was used to run the application

#### Instructions to run the application
1. User is asked to provide which operator 

![Simple Calculator](Calculator1.png)

2. User is then asked to enter a number and the programme will provide a timestable for that number from 1-15
![Simple Calculator](Calculator2.png)

3. Programme can still be ran and the user has the choice of selecting a different operator and a different number
![Simple Calculator](Calculator3.png)

4. Programme will then provide the output based on what the user has selected
![Simple Calculator](Calculator4.png)
